# 单词发音播放器

这是一个简单易用的单词发音播放器，帮助用户学习英语单词的发音和含义。

## 功能特点

- 支持多个单词列表分类
- 单词发音功能，使用Web Speech API
- 可设置单词播放次数
- 支持单词列表的创建、删除和切换
- 支持单词导入功能
- 支持PWA，可安装到设备上使用

## 项目结构

## manifest.json 文件说明

manifest.json 是 Progressive Web渐进.webmanifest 文件的简写，是 Progressive Web App (PWA) 的配置文件。它是一个JSON格式的文件，告诉浏览器关于你的Web应用的信息，以及当用户安装应用时应该如何表现。

### 主要作用

1. **应用标识**：定义应用的名称、描述等基本信息
2. **视觉外观**：指定应用的图标、颜色主题等
3. **行为控制**：定义应用启动时的行为，如启动URL、显示模式等
4. **安装提示**：浏览器使用这些信息来提供"添加到主屏幕"的提示

### 关键配置项说明

- `name`: 应用的完整名称
- `short_name`: 应用的简称，用于空间有限的地方（如手机主屏幕）
- `start_url`: 应用启动时打开的URL
- `display`: 显示模式，`standalone`表示应用将以独立应用的形式运行，有自己的窗口
- `background_color`: 应用启动时的背景颜色
- `theme_color`: 应用的主题颜色，影响浏览器UI元素的颜色
- `icons`: 应用的图标集合，不同尺寸用于不同设备
- `description`: 应用的简短描述
- `orientation`: 应用的首选方向

### 如何使用

当用户访问你的Web应用时，现代浏览器会检查是否存在manifest.json文件。如果存在，浏览器会：

1. 读取文件中的配置信息
2. 提供"添加到主屏幕"的选项
3. 使用指定的图标和名称创建桌面快捷方式
4. 当用户点击快捷方式时，以指定的显示模式打开应用

## PWA功能说明

PWA (Progressive Web App) 是一种使用现代Web技术创建的应用，它具有以下特点：

1. **可安装性**：用户可以将应用安装到设备主屏幕
2. **离线访问**：即使没有网络连接，应用也能正常工作
3. **类似原生应用的体验**：全屏运行，有自己的图标和启动画面
4. **自动更新**：应用会自动更新到最新版本

这个单词发音播放器已经配置为PWA，用户可以将其安装到手机或电脑上，享受更好的学习体验。

## 数据结构说明

### default-lists.js

该文件包含了所有默认的单词列表数据，结构如下：

```javascript
const DEFAULT_LISTS = {
  "列表名称": [
    { english: "单词", phonetic: "音标", chinese: "中文含义" },
    // 更多单词...
  ],
  // 更多列表...
};
```

目前包含的默认列表有：
- 默认列表
- 动物列表
- 颜色列表
- 数字列表
- 食物列表
- 水果列表
- 交通工具列表
- 家庭成员列表
- 学校列表
- 职业列表

### localStorage数据结构

应用使用localStorage存储用户数据，结构如下：

```javascript
{
  lists: {
    "列表名称": [
      { english: "单词", phonetic: "音标", chinese: "中文含义" },
      // 更多单词...
    ],
    // 更多列表...
  },
  currentList: "当前选中的列表名称",
  defaultPlayTimes: 默认播放次数
}
```

## 如何使用

1. 打开index.html文件即可使用
2. 点击"播放发音"按钮播放当前单词的发音
3. 点击"按默认次数播放"按钮按设置的次数播放当前单词
4. 点击"单词列表"按钮查看和管理单词列表
5. 点击"列表管理"按钮创建、删除和切换单词列表
6. 点击"导入单词"按钮导入自定义单词列表

## 如何添加新的默认列表

1. 打开default-lists.js文件
2. 在DEFAULT_LISTS对象中添加新的列表，格式如下：
   ```javascript
   "新列表名称": [
     { english: "单词1", phonetic: "音标1", chinese: "中文含义1" },
     { english: "单词2", phonetic: "音标2", chinese: "中文含义2" },
     // 更多单词...
   ],
   ```
3. 保存文件即可，应用会自动识别新添加的列表
